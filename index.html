Configurar sincronização

Para que todo o time veja os mesmos dados em tempo real, conecte um banco Firebase gratuito. Leva 3 minutos.

1

Acesse console.firebase.google.com, crie um projeto gratuito e ative o Realtime Database no modo de teste.

2

Em Configurações do projeto → Seus apps, copie o apiKey e a databaseURL.

3

Abra o arquivo assets/app.js e preencha o objeto FIREBASE_CONFIG com todos os valores. Depois suba no GitHub.

Como o time todo acessa

O app está no GitHub Pages. Para sync em tempo real, configure o Firebase (gratuito). Depois basta compartilhar a URL.

1

Configure o Firebase

Abra assets/app.js e preencha FIREBASE_CONFIG com os dados do seu projeto Firebase.

2

Suba no GitHub

Faça commit e push. O GitHub Pages publica automaticamente em poucos segundos.

3

Compartilhe a URL

A URL do GitHub Pages fica no formato usuario.github.io/repositorio. Mande no grupo do time.

4

Sync automático

Qualquer alteração feita por qualquer pessoa aparece para todo o time em tempo real, sem precisar atualizar a página.

💡 Sem Firebase? O app funciona normalmente, mas os dados ficam salvos apenas no navegador de cada pessoa.
